# 📊 Interactive Sales Reporting Dashboard in Excel

## 📌 Project Overview
The Interactive Sales Reporting Dashboard is an Excel-based business intelligence project designed to analyze and visualize sales performance across different brands and time periods. The dashboard enables users to monitor monthly sales trends, compare yearly performance, and gain actionable insights through interactive reports and visualizations.

## 🎯 Objectives
- Analyze sales data across multiple brands and years.
- Track monthly sales performance and growth trends.
- Create interactive reports for better decision-making.
- Visualize key business metrics using charts and dashboards.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- SUMIFS
- Data Validation
- Conditional Formatting

## 📈 Key Features
- Brand-wise sales analysis
- Monthly sales trend visualization
- Year-over-year performance comparison
- Interactive dashboard with filters
- KPI tracking and reporting
- Dynamic charts and reports

## 📊 Dashboard Insights
- Identified top-performing brands and months.
- Compared sales performance across different years.
- Monitored sales growth patterns and seasonal trends.
- Generated business insights through interactive visual reports.
